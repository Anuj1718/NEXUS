In CSS, the `font-weight` property specifies the weight (or boldness) of the font. It allows you to control how thick or thin the characters appear in your text. **Syntax:** selector { font-weight: value; } **Values:** 1. **Keyword Values:** - `normal`: The default font weight (usually equivalent to 400). - `bold`: A bold font weight (usually equivalent to 700). - `bolder`: A bolder weight than the parent element's font weight. - `lighter`: A lighter weight than the parent element's font weight. 2. **Numeric Values:** - Numeric values range from 100 to 900, where: - `100` is the lightest weight. - `400` is equivalent to `normal`. - `700` is equivalent to `bold`. - Values in between (e.g., `200`, `300`, `500`, `600`, `800`, `900`) can be used for more granular control over font weight, depending on the font family. **Examples:** /* Using keyword values */ p { font-weight: normal; /* Default weight */ } /* Using bold */ h1 { font-weight: bold; /* Bold text */ } /* Using numeric values */ h2 { font-weight: 600; /* Semi-bold text */ } /* Using lighter and bolder */ p { font-weight: lighter; /* Lighter than the parent element */ } **Best Practices:** - **Consistency:** Use consistent font weights across your design to maintain a cohesive look. Avoid using too many different weights, as it can create visual clutter. - **Readability:** Ensure that the font weight you choose enhances readability. For body text, a normal or slightly bold weight is often best, while headings can use bolder weights for emphasis. 4) font-style: In CSS, the `font-style` property is used to specify the style of the font, such as whether it is normal, italic, or oblique. This property allows you to add emphasis to text, making it visually distinct from the surrounding content. **Syntax:** selector { font-style: value; } **Values:** 1. **Keyword Values:** - `normal`: The default font style, which displays text in a standard manner without any special emphasis. - `italic`: Displays text in an italicized style, which is typically slanted to the right. - `oblique`: Similar to italic, but may not be specifically designed as an italic font. It usually slants the text to the right but can appear less stylized than true italics. 2. **Inherited Values:** - `inherit`: Inherits the font style from the parent element. - `initial`: Sets the property to its default value (normal). - `unset`: Resets the property to its natural value, which is either inherited or initial depending on the context. **Examples:** /* Using normal font style */ p { font-style: normal; /* Standard text */ } /* Using italic font style */ h1 { font-style: italic; /* Italicized heading */ } /* Using oblique font style */ h2 { font-style: oblique; /* Oblique heading */ } /* Using inherit */ span { font-style: inherit; /* Inherits the font style from the parent element */ } /* Using initial */ div { font-style: initial; /* Resets to the default font style (normal) */ } /* Using unset */ blockquote { font-style: unset; /* Resets to the natural value, which is normal if not inherited */ } **Best Practices:** - **Emphasis:** Use `italic` or `oblique` sparingly for emphasis, as overuse can reduce readability. Reserve these styles for specific cases, such as quotes or citations. - **Consistency:** Maintain a consistent use of font styles across your design. Mixing too many styles can create visual confusion. - **Accessibility:** Ensure that italicized or oblique text remains legible, especially for users with visual impairments. Some fonts may not render well in italic or oblique styles, so test your design across different devices and browsers. 5) font-variant: (HW) In CSS, the `font-variant` property is used to control the use of small-caps, ligatures, and other typographic features in a font. This property allows for enhanced text styling, particularly in professional or artistic contexts. **Syntax:** selector { font-variant: value; } **Values:** 1. **Keyword Values:** - `normal`: The default font variant, which displays text in its standard form without any special features. - `small-caps`: Displays text in small capital letters, where lowercase letters are rendered as smaller uppercase letters. - `all-small-caps`: Similar to `small-caps`, but all letters, including uppercase ones, are rendered as small capitals. - `petite-caps`: Displays text in petite capitals, which are smaller versions of uppercase letters. - `all-petite-caps`: Similar to `petite-caps`, but all letters are rendered as petite capitals. - `unicase`: Displays text in a mixed case where both uppercase and lowercase letters are used, but with a consistent height. - `titling-caps`: Displays text in titling capitals, which are typically larger and more decorative uppercase letters used for titles or headings. 2. **Ligatures:** - `common-ligatures`: Enables common ligatures in the font, such as "fi" and "fl". - `no-common-ligatures`: Disables common ligatures. - `discretionary-ligatures`: Enables discretionary ligatures, which are stylistic variations of certain letter combinations. - `no-discretionary-ligatures`: Disables discretionary ligatures. - `historical-ligatures`: Enables historical ligatures, which are older forms of letter combinations. - `no-historical-ligatures`: Disables historical ligatures. 3. **Contextual Alternates:** - `contextual`: Enables contextual alternates, which allow the font to adjust letterforms based on their surrounding characters. - `no-contextual`: Disables contextual alternates. **Examples:** /* Using normal font variant */ p { font-variant: normal; /* Petite capital letters */ } /* Using all-petite-caps font variant */ h4 { font-variant: all-petite-caps; /* All letters in petite capitals */ } /* Using unicase font variant */ p { font-variant: unicase; Ligatures: In typography, ligatures are special characters that combine two or more letters into a single glyph to improve the appearance and readability of text. Common examples include "fi", "fl", and "ff". Ligatures are often used to prevent awkward collisions between letters and to create a more visually pleasing flow in certain fonts.

Petite Capitals Explanation:

Petite Capitals: Petite capitals are a typographic feature similar to small capitals, but they are even shorter in height. They are designed to match the x-height of lowercase letters, making them more harmonious when used within text. Petite capitals are mainly available in professional or advanced typefaces that support this feature.
